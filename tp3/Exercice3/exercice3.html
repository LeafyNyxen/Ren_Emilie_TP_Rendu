Ce que le TP ma apporter

Le code dans l'exercice 3 a pour but de concevoir un menu hamburger. On nous demandait de réarranger les éléments pour qu'ils soient accessibles sur le bureau tout en se transformant en menu responsive sur mobile. Les connaissances principales sont la gestion des Media Queries en plus du DOM via JavaScript pour orchestrer l'ouverture et la fermeture du menu. Pour que cela fonctionne nous utilisons une transition CSS utilisant max-height et opacity pour créer un effet de glissement fluide. Le niveau de difficulté est de ⭐⭐⭐⭐. C'est l'un des exercices les plus complets car il demande une maîtrise combinée du HTML, du CSS pour l'animation et du JavaScript pour la logique mais aussi complexe.
