Ce que le TP ma apporter

Le code dans l'exercice 1 a pour but de nous mettre à l'épreuve sur l'adaptation dynamique d'une interface on nous demandait de créer un bloc visuel dont les propriétés changent selon une certaine dimension. Les connaissances principales sollicitées ici sont les Media Queries (@media), qui permettent de définir des points des breakpoints, ainsi que le Flexbox sur l'ensemble du document. Par défaut, le bloc est bleu, mais il change en orange sous les 960px, puis en rouge sous les 480px. Le niveau de difficulté est de ⭐ c'est un exercice simple qui ne demande pas de si grandes connaissances en css mais demande surtout une maîtrise plutôt correcte de la logique en CSS.
